# Testing nhojpatrick archetypes

```bash
$ mvn archetype:generate \
	-DarchetypeGroupId=com.github.nhojpatrick.archetypes \
	-DarchetypeArtifactId=nhojpatrick-archetype-TYPE-project \
	-DarchetypeVersion=VERSION \
	-DgroupId=TESTING_GROUPID \
	-DartifactId=TESTING_ARTIFACTID \
	-Dversion=TESTING_VERSION
```
